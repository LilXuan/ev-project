This project focuses on stimulating a data scale up processing pipeline.

## Technologies Used
- Language: PySpark, Bash
- Storage: Hadoop HDFS
- Compute engine: Spark
- Mornitoring: Prometheus, Grafana
- Infrastructure: VMware Workstation

## Features
- Built an end to end ML pipeline to apply big data
- Ingesting data, processing data and training model
- Mornitoring followed 3 layers: OS metrics, Spark metrics, and applications metrics

## Current Progress
- Collect metrics and visuallizing dashboard
- Expand monitoring dashboards for Spark executor and JVM metrics
- Analyze and explain results

## Future Improvements
- Add containerized deployment using Docker
- Explore workflow automation and CI/CD integration
